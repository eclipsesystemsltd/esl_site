Skills

At Eclipse Systems we are not tied to particular technologies and believe in choosing the most suitable technology to achieve the desired objective.

Our staff are trained to understand technology at a fundamental level and will apply good design techniques and practices regardless of the technologies being employed.

If your project requires a specific technology that’s no problem – we’ve got plenty of experience of the main ones, and many of the lesser ones as well. If you need help with something unusual, fear not. We are very good at mastering new (and sometimes old!) technologies and choose our staff for this level of adaptability.

For a demonstration of our capabilities please get in touch.

.NET

More

.NET is Microsoft’s premier platform for creating applications for web, mobile and desktop, typically using the C# or VB.NET languages. Created in the image of Java, the .NET platform shares many of the same benefits and adds plenty of its own.

At Eclipse Systems we have been working with .NET since it first appeared in 2002 and have successfully implemented many .NET projects.

Java

More

Java remains a number one choice across the industry, due to its maturity, power and flexibility. A huge amount of robust, well supported functionality is available at no cost to you.

The Java ecosystem is all about choice, and a successful Java project often draws together multiple frameworks into an integrated whole. Choosing between the many possibilities and integrating them successfully is a key challenge and one that we are highly experienced in tackling, having successfully delivered enterprise Java systems for a vast and diverse range of clients.

Mobile

More

Eclipse Systems can provide custom mobile application development and consultancy on any of the following phone and tablet platforms.

  • Apple – iPad, iPhone, iPod Touch
  • Android
  • BlackBerry
  • Windows Phone
  • Windows Mobile
  • Symbian

Best results are typically achieved by using the native tools and frameworks provided by the vendor for each mobile platform. We have a deep understanding of these and can deliver a highly polished result tailored to your chosen platform or platforms.

Cross-platform App Development

Where cost is an issue we can also use HTML-based approaches or cross-platform frameworks to target multiple platforms at once.

Web Technologies

More

Producing a modern, highly functional website with a responsive user interface requires mastery of a wide range of technologies that run in the user’s web browser, not just HTML.

Eclipse Systems possesses extensive experience with all the key client-side web technologies needed to produce a sophisticated website – notably:

  • HTML and latest CSS standards and techniques
  • JavaScript and the DOM
  • HTML 5
  • jQuery and other JavaScript libraries
  • Flash
  • Flex
  • Silverlight

We are also very familiar with the latest techniques for optimising website speed and performance, which can make a critical difference to the success of your online business.

[Image courtesy of kromkrathog / FreeDigitalPhotos.net]

Ruby On Rails

More

Ruby on Rails was once the upstart technology that kick-started a mini revolution in web application development, but these days it is relatively mainstream and a low-risk choice. Ruby on Rails provides standard solutions to common requirements and, if you stick to its metaphorical rails, you can achieve a great deal with only a small amount of effort.

If you go beyond the simple use-cases that’s when things become more challenging. Ruby is a sharp tool with a steep learning curve, so Eclipse Systems’s extensive Ruby experience and our superb technical pedigree are key to us using it effectively on your projects.

Other

More

At Eclipse Systems we take great pride in our breadth and depth of experience. It lets us use the best tool for the job, and apply ideas and techniques from across the technology spectrum to your specific problem.
We have built up expertise in languages and frameworks too numerous to list. Here are just a few of the other skills we can supply.

  • PHP
  • Python
  • C/C++
  • Drupal – a PHP-based content management framework
  • Groovy – a language similar to Java but bringing in many of the benefits of Ruby and Python
  • Grails – the predominant web framework for Groovy development, built on top of Spring, Hibernate and other Java web staples
