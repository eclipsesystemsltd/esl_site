Typical PDP-8/e Configuration

The PDP-8/e contains a 40 slot Omnibus backplane and the cards shown here represent a typical complement of cards, although the EAE option may not have been so common as I haven't seen many for sale. This configuration includes the M837 that is needed to support more than 4K of memory.

The cards are shown in the order of installation, but note that the first slot would contain the programmers console.

  • M8300

    KK8E Major Registers


  • M8310

    KK8E Major Registers Control


  • M8340

    KE8E EAE Decoder & Step Counter


  • M8341

    KE8E EAE Multiplexers & Timing


  • M8330

    KK8E Timing Board (System Clock)


  • M8650

    KL8E Asynchronous Data Control


  • M837

    MC8E Extended Memory Control


  • M849

    KK8E Memory RFI Shield


  • G111

    MM8EJ H212 8K Memory Sense Inhibit


  • G646C

    MM8EJ H212 8K Core Memory


  • G233

    MM8EJ H212 8K Memory XY Drivers


  • M8320

    KK8E Bus Loads
